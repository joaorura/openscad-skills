# OpenSCAD Skills Collection

This repository contains a collection of Agent Skills for AI coding assistants. These skills teach agents how to properly structure, plan, and execute modern OpenSCAD modeling tasks using the intent-based BOSL2 library and the data-driven Boardgame Insert Toolkit (BIT).

> **Disclaimer:** This repository and all included skills were generated by AI.

## Included Skills

- **openscad-project-understanding**: Organization and modularity guidelines.
- **openscad-workflow-planning**: Parametric design logic and library selection.
- **openscad-bosl2-mastery**: The BOSL2 intent-based attachment system and syntax.
- **openscad-boardgame-organizer**: Data-driven design using The Boardgame Insert Toolkit (BIT).

## Installation

Agent skills are designed to be loaded by your CLI agent.

### Claude Code
1. Clone this repository or download the .skill files from the \dist/\ folder.
2. In your terminal, run the Claude Code skills command:
   \\\ash
   claude skills install path/to/skill-name.skill
   \\\

### OpenClaude
1. Download the \.skill\ files from the \dist/\ folder.
2. Install them using the OpenClaude CLI:
   \\\ash
   openclaude skills install path/to/skill-name.skill
   \\\

### Gemini CLI
1. Download the \.skill\ files from the \dist/\ folder.
2. Install them either locally (workspace scope) or globally (user scope):
   \\\ash
   gemini skills install path/to/skill-name.skill --scope workspace
   # OR
   gemini skills install path/to/skill-name.skill --scope user
   \\\
3. In an interactive session, type \/skills reload\.

### Copilot CLI
1. Copilot CLI typically auto-discovers skills from installed plugins or specific configuration paths. Refer to your Copilot CLI documentation for mapping standard \.skill\ packages or folder structures.

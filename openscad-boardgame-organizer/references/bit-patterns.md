# BIT Pattern Reference

## Constant Types
- `BOX_SIZE_XYZ`: [width, depth, height]
- `BOX_LID`: [style, pattern, label]
- `LID_STYLE_SLIDING`, `LID_STYLE_CAP`, `LID_STYLE_INSET`

## Feature Types
- `FEATURE_GRIDS`: Uniform compartment arrays.
- \`FEATURE_DIVIDERS\`: Custom internal walls.
- \`FEATURE_LABEL\`: Text labels on the box or lid.

## Ergonomic Constants
- \`LABEL_STYLE_INTERNAL\`: Engraved text.
- \`LABEL_STYLE_EXTERNAL\`: Raised text.
